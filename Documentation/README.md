# Africa Disease Dashboard 2019-2024

## Project Overview

This project presents an interactive Power BI dashboard analyzing disease patterns across Africa between 2019 and 2024.

The dashboard provides insights into:

- Disease trends over time
- Regional disease burden
- Cases and mortality patterns
- Case Fatality Rate (CFR)
- High-risk locations
- Public health indicators

## Business Objective

The objective of this project is to support healthcare decision-makers by transforming disease data into actionable insights for:
- Resource allocation
- Disease prevention planning
- Public health interventions
- Monitoring health outcomes

## Tools Used

- Power BI
- Power Query
- DAX
- Excel/CSV
- Data Cleaning & Transformation

## Data Process

1. Imported raw disease dataset

2. Cleaned missing values and inconsistent records

3. Standardized country and disease categories

4. Built calculated measures using DAX

5. Developed interactive dashboard
